# TesteBancoPan

## Features

- Entidades:
  - Pessoas (Fisica e Juridica)
  - Endereco

- API CRUD que contém: :heavy_check_mark:
  - Pessoas (Fisica e Juridica)
    - Criar pessoa;
    - Listar pessoas;
    - Atualizar pessoa;
    - Deletar pessoa.
